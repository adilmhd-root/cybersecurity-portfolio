# Adil Muhammad | Professional Portfolio

A high-performance, dark-themed portfolio website designed for a modern cybersecurity analyst. Features a sophisticated "Bento Grid" layout, custom interactive cursor, and an animated "Attack → Defense Flow" methodology diagram.

## Key Features

- **Modern Aesthetic**: Dark obsidian theme with electric blue and violet accents.
- **Bento Grid Layout**: Trendy, box-based layout for the About and Skills sections.
- **Attack-Defense Flow**: An animated, interactive diagram demonstrating the cybersecurity methodology from reconnaissance to mitigation.
- **Custom Interactions**: 
  - Custom cursor follower.
  - Scroll-triggered reveal animations.
  - Glassmorphism on navigation.
- **Responsive**: Fully adapted for Mobile, Tablet, and Desktop.

## Technical Stack

- **HTML5**: Semantic structure.
- **CSS3**: Variables, Flexbox, Grid, Animations.
- **JavaScript**: IntersectionObserver API for performance-friendly scroll animations.

## Customization Guide

1. **Content**: Edit `index.html` to update text, links, and project details.
2. **Colors**: Modify the `:root` variables at the top of `style.css` to change the primary (`--accent-primary`) and secondary (`--accent-secondary`) colors.

## How to Run

Open `index.html` in Chrome, Edge, Safari, or Firefox.
